微信验证控制台

把公众号接入 SaaS，用微信 OpenID 锁定免费额度领取资格。

推荐操作顺序

三步完成公众号验证接入

  1. 1保存公众号配置
  2. 2复制 Webhook 到微信
  3. 3查询用户验证结果

步骤 1

先把三个系统连起来

这里只做一件事：告诉本服务“你是谁、微信是谁、你的 SaaS 接口在哪里”。

1. 这条接入叫什么？ 给这次接入起一个内部名字。它会出现在微信 Webhook URL 里，建议用英文小写。
2. 微信公众号信息 从微信开发者平台复制。AppID、Token、AppSecret 必须和微信侧配置一致。
3. 你的 SaaS 怎么和本服务通信？ 这两个值就是 API Key。你的 SaaS 后端调用本服务创建验证会话、查询绑定状态时使用。
4. 让流程真正闭环 用户在公众号发账号后，本服务先问你的 SaaS 能不能绑定；完成后再把结果通知你的 SaaS。 看 SaaS 回调端实现示例（Node.js / Python / Go / PHP） →
查看或编辑已保存租户

步骤 2

复制到微信

把这个地址填到微信开发者平台的「消息推送 URL」。微信会把关注和用户消息发到这里。

当前 Webhook URL /wechat/{tenantId}/webhook
