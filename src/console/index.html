接入教程

  1. 登录微信公众号后台，进入开发者配置。
  2. 复制 `AppID` 和 `AppSecret` 到左侧表单。
  3. 生成一串随机 `Token`，同时填入公众号后台和左侧表单。
  4. 公众号服务器 URL 填写：/wechat/{tenantId}/webhook
  5. V1 选择明文模式，`EncodingAESKey` 先保存备用。
  6. SaaS 后端先创建待绑定会话，再引导用户关注公众号并发送平台账号。
